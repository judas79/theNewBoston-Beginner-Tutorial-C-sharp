To allow th user to choose a color for the font, we will set the font color property to true, or false if you want to stop
the user from choosing one.  The color will not be able to be displayed in the textBox1, until textBox1s, properties are
set to allow color change.  We will set textBox1 to color enabled using fontdialog within the if() statement.


        // L83 FontDialog is global name
        FontDialog IFD;
        private void button1_Click(object sender, EventArgs e)
        {
            // L83 create instance of FontDialog
            IFD = new FontDialog();

            // L83 set max and min size using properties, for size
            IFD.MinSize = 10;
            IFD.MaxSize = 18;

            // L83 enable the user to choose the font color from fontdialog by showing the color picker
            IFD.ShowColor = true;

            // L83 verify user has selected somehting
            if (IFD.ShowDialog() == System.Windows.Forms.DialogResult.OK)
            {
                // L83 set font in textBox1 to fontdialog selected
                textBox1.Font = IFD.Font;

                // L83 set textBox1 to show font (foreground) colors selected from colordialog
                textBox1.ForeColor = IFD.Color;
            }



We will also enable the help button, so the help button shows, in the FontDialog.  We will also have to set up a function,
as we did in the previous lesson, to enable the help button to do something when its clicked.  To enable showing the help

            // L83 show the help button, in FontDialog window.

To have the help button do something when an event, like a click, triggers it, we will use the helpRequest method.  

            // L83 show the help button, in FontDialog window.
            IFD.ShowHelp = true;

            // L83 enable the help button to do something when help is clicked (event)
            IFD.HelpRequest += IFD_HelpRequest;
